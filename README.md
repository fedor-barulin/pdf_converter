# Enterprise Document Converter 🚀

Мощный движок уровня Enterprise для конвертации смешанных PDF, DOC, DOCX документов (сканы, таблицы, текст) в структурированный JSON формат с чистым контентом. Решение полностью избавлено от лишних оберток и поисковой инфраструктуры, представляя собой идеальный Data Extraction Pipeline.

---

## 🏗️ Архитектура: Document Pipeline

Проект выполняет только одну строго заданную функцию: **конвертация документов в структурированный JSON**.

**Pipeline Архитектура:**
`Document → Parser → OCR (если нужно) → Layout detection → Structure extraction → Table extraction → Noise removal → Deduplication → Chunking → JSON export`

### 🔍 Ключевой функционал:
1. **Многоформатность:** Поддержка PDF, `DOC`, `DOCX` (С автоматической конвертацией старых форматов через Python win32com).
2. **Распознавание сканов (OCR):** Автоматическое извлечение текста через Tesseract OCR, если PDF содержит изображения.
3. **Advanced Smart Cleaner (Удаление мусора):**
   - Детектирует и удаляет "сквозные" колонтитулы.
   - Очищает признаки даты выгрузки, счетчики страниц, служебные подписи, watermarks.
   - **Deduplication:** удаление повторяющихся строк и параграфов.
4. **Извлечение структуры и таблиц:**
   - Разбиение документа на `sections`, `subsections`, `paragraphs` и списки.
   - Извлечение таблиц с сохранением строк и колонок (из PDF и DOCX).
5. **Поиск URL статьи:** Автоматическое обнаружение ссылки статьи в header/footer или среди основного текста (добавляется в поле `article_url`).
6. **Semantic Chunking:** Разбиение очищенного текста на чанки размером 500-800 токенов (с сохранением в результирующем файле для будущего использования).

---

## ⚙️ Выходной Формат

Проект генерирует единый чисто структурированный **JSON**-файл. Генерация markdown и txt отключена.

```json
{
  "title": "Название статьи",
  "article_url": "https://site.com/article/123",
  "content": "очищенный текст",
  "sections": [],
  "tables": [],
  "chunks": [],
  "metadata": {
    "source_file": "document.pdf",
    "pages": 12,
    "language": "ru"
  }
}
```

## 🚀 Запуск и Интеграция

### 🐳 Запуск через Docker

Самый быстрый способ запустить проект — использовать Docker Compose. Все зависимости (OCR, Python, Node.js) уже настроены.

```bash
docker-compose up --build
```

Подробная инструкция по настройке и командам Docker находится в файле [DOCKER_INSTRUCTIONS.md](./DOCKER_INSTRUCTIONS.md).
